# Фінальний проєкт: Функція екзамену для онлайн-курсу

Цей проєкт реалізує нову функцію оцінювання (екзамен) для платформи онлайн-курсів на базі фреймворку Django.

## Реалізований функціонал:

1.  **Моделі даних:**
    * Створено моделі `Question` (Питання) та `Choice` (Варіанти відповідей) для наповнення екзамену.
    * Створено модель `Submission` (Подання) для відстеження спроб студентів та збереження обраних відповідей.

2.  **Інтерфейс адміністратора:**
    * Налаштовано Django Admin для керування питаннями.
    * Використано `StackedInline` для додавання варіантів відповідей безпосередньо на сторінці редагування питання.

3.  **Представлення (Views) та Логіка:**
    * Реалізовано функцію `submit` для обробки POST-запитів форми екзамену.
    * Реалізовано функцію `show_exam_result` для автоматичного підрахунку балів шляхом порівняння вибраних відповідей із правильними.

4.  **Шаблони (Frontend):**
    * Оновлено шаблон деталей курсу для відображення блоку з екзаменом.
    * Створено сторінку результатів, яка показує підсумковий бал, статус (Склав/Не склав) та детальний розбір відповідей.

## Використані технології:
* Python 3, Django
* HTML5, CSS3 (Bootstrap 4)
* SQLite
